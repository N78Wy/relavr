# relavr / relavr-view 设计语言

## 设计定位

relavr 与 relavr-view 的整体视觉应保持统一，风格上强调 **简洁、现代、科技感、易识别**。
整体不走复杂装饰路线，而是以 **圆润的几何造型、清晰的功能表达、轻微立体与光感** 来建立品牌识别。

这套设计语言适用于：

* App icon
* 启动页
* 登录页
* 设置页
* 空状态插画
* 品牌宣传图
* 官网或介绍页中的产品视觉延展

## 风格关键词

**简洁、科技、轻立体、圆润、清爽、统一、友好、现代**

## 视觉特征

整体图形应以 **大圆角、几何化、中心构图** 为基础。
通过适度的渐变、高光和阴影，营造轻量的数字科技感。
视觉表现上避免厚重、复杂、写实，更强调“清晰可识别”和“轻盈有层次”。

## 色彩方向

品牌主色以 **湖蓝、青蓝、蓝绿色** 为核心。
颜色使用上以同色系渐变为主，突出科技感和空间感。
深色用于承载屏幕、镜片、阴影等视觉重点，浅色用于高光、边缘和发光感表现。
整体配色应尽量克制，避免过多颜色同时出现，保证画面统一。

## 造型语言

所有核心元素应尽量保持：

* 轮廓简洁
* 圆角明显
* 线条柔和
* 结构稳定
* 容易识别

图形不追求复杂细节，而是通过简化后的设备轮廓和功能符号来传达含义。
例如 VR 眼镜、显示器、播放按钮、无线信号等，都应以更简洁直观的方式出现。

## 材质与光影

整体采用 **轻拟物 + 数字玻璃感** 的视觉方式。
可以使用少量高光、柔和阴影、透明感和发光感来增强层次，但应避免过强反光、金属质感或真实纹理。
重点是让界面和图形看起来“干净、现代、有科技感”。

## 图形元素建议

在后续设计中，可以围绕以下元素形成统一识别：

* VR 眼镜：代表发送端、设备端
* 屏幕 / 播放符号：代表接收端、显示端
* 无线信号 / 投屏符号：代表连接与传输
* 光晕 / 波纹 / 抽象背景层次：代表数字内容与空间流动
* 圆角卡片：作为 UI 和插画中的基础容器

## 排版语言

排版应保持简洁、直接、清晰。
字体建议选择现代无衬线字体，整体偏干净、中性、易读。
标题和功能信息要有清晰层级，避免过度装饰。
品牌名和关键功能文字应优先保证识别性与阅读效率。

## UI 延展原则

在界面设计中，应延续图标中的视觉语言：

* 使用足够留白
* 强调卡片化布局
* 采用圆角按钮和圆角面板
* 控制信息密度
* 使用少量重点色突出状态和操作
* 动效以柔和、稳定、轻盈为主

整体体验上要让用户感受到：
**简单、直观、专业、可靠**

## 双产品区分原则

虽然 relavr 与 relavr-view 共享同一套设计语言，但视觉重点可以略有区分：

**relavr**
更强调“发送、连接发起、设备源头”。
视觉中心可以偏向 VR 设备本体、扫描连接、推送动作。

**relavr-view**
更强调“接收、显示、内容呈现”。
视觉中心可以偏向显示界面、播放状态、接收反馈和投屏结果。

## 一句话总结

**这是一套以蓝绿色科技感为核心、以圆润几何和轻立体表现为基础、强调连接与显示场景的统一视觉语言。**
